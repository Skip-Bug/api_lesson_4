# Telegram BOT

## Что нужно сделать перед запуском
Проверьте версию Python:
   ```bash
python --version
   ```
- Должна быть версия 3.8 или выше

Установите необходимые библиотеки:
   ```bash
pip install -r requirements.txt
  ```

**Часть настроек проекта берётся из переменных окружения.**   
**Чтобы их определить, создайте файл .env (точка в начале обязательна!) рядом с main.py**
**и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.**
```
NASA_ID=API-ключ для работы с API NASA
TG_BOT_TOKEN=ключ для работы телеграмм Бота
PROXY_URL= Прокси для работы Бота в России(http://127.0.0.1:порт_прокси) если не нужен оставте пустым
TG_CHANNEL_ID=Имя канала или его номер например (-1003711383499)
```

### Получите токен `NASA_ID`
- Зарегистрируйтесь на сайте [NASA](https://api.nasa.gov/)
- Найдите там свой **API ключ** (обычно в личном кабинете или настройках профиля)
- Скопируйте его

### Иснтрукция регистрации для [Бота](https://way23.ru/%D1%80%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F-%D0%B1%D0%BE%D1%82%D0%B0-%D0%B2-telegram.html)
И получения его `TG_BOT_TOKEN` например 958423683:AAEAtJ5Lde5YYfkjergber
Также понадобится канал `TG_CHANNEL_ID` в [telegram](https://smmplanner.com/blog/otlozhennyj-posting-v-telegram/#01) и его название: 
публичный пишут @мой_канал, а частный (-1003711383499)


## Как запустить?
**Качаем наш код**
**Запускаем CLI** Нажмите Win + R, введите cmd
```bash
   cd укажите_путь_к_папке_api_lesson_4
```
Для начало заполните папку **images** с помощью:
```bash
python any_link.py
```
или
```bash
python spacex.py
```
или
```bash
python nasa_apod.py
```
или
```bash
python nasa_epic.py
```

## any_link.py

Сохраняет картинку по URL в указанную папку, поддерживает несколько ссылок через **,**.

## spacex.py

Получает изображения запусков SpaceX-API и сохраняет их.   
Для скачивания нужны ID запусков. Если не вводить ID просто скачает изображения с последнего запуска.   
```bash
python spacex.py 5eb87d47ffd86e000604b38a
```

## nasa_apod.py

Получает изображения "Дня" с NASA APOD API.   
Для работы требуется API-ключ, но если его не указать будет использован демо ключ.   
Имет несколько настроек:   
Количество снимков от 1 до 100, если не указать можно выбрать определенный день.   
Также есть выбор качества изображения HD-версия или обычная.   

## nasa_epic.py

Получает изображения "Земли" с NASA EPIC API   
Для работы требуется API-ключ, но если его не указать будет использован демо ключ.   
Можно выбрать день, если не выбрать день то за последнюю доступную дату    
(обычно текущая дата, но если на сегодня ещё нет снимков, то за последний день, когда они есть.   

## main.py

Для работы бота
**Запустите main.py**
```bash
python main.py -h
```

Скрипт запускает телеграмм Бота , котрый с задержкой в 4 часа   
или заданной пользователем опубликует изображения из папки images.   
Папка images должна содержатьтолько изображения!!!   
Когда все изображения отправлены Бот перемешает всё и начнет заного.   
Если удалить все изображения Бот будет ждать загрузки новых файлов.   
Ctrl+C остановит работу Бота.
